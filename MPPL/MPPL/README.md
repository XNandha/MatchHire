# MatchHire — Prototype

This repository contains a lightweight static prototype structure for MatchHire, a job-matching platform for students and SMEs.

## Structure
- index.html - landing page
- login.html / register.html - authentication pages
- job-seeker/ - pages for job seekers
- employer/ - pages for employers
- admin/ - admin dashboard pages
- assets/css - styles
- assets/js - client-side scripts
- api/ - placeholder API files (server implementation required)

## How to run
Open `index.html` in your browser or serve with `python -m http.server` from the project root for basic testing.

## Notes
This is a static prototype meant for student teams. Replace placeholder APIs with a backend implementation for production.
